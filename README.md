# Lab 1

## Download & Run

1. Clone this repository

```bash
git clone https://github.com/future-stardust/ip-0x-lab-1-MrPaschenko
```

2. Install necessary packages

```bash
npm i
```

3. Run program

```bash
node index
```

4. Run tests

```bash
npm test
```
